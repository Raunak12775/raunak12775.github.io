What is Quarto?

Quarto is a markup language which can be used to generate beautiful technical or scientific documents.

What can be done with it?

Many things can be done like:

  • Blogs/Websites
  • Articles
  • Presentations
  • Book

and also one can translate with different kinds of document format like html, pdf, pptx, docx, odt etc. with an ease of simple commands. For more I recommend you to check out the official Quarto website.
